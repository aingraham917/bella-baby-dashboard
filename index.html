🔧 Delivery Standards Review

📋 Bella Baby Delivery Standards tap to expand ▼
  • 18–20 Images (28–30 for Twins)
  • Newest Intro/Outro Slides
  • Copyright Slide (if gallery isn't purchased)
  • Image Aspect Ratio (8×12 or 4×6)
  • Image File Size (3MB–8MB)
  • 3 Notes
  • Metadata Copyright
  • Proper Capitalization/Spelling for Emailing Purposes
  • Baby Name: "Baby" or proper name for Child's First Name. Last name should also be present, OR a note about why it was not.
  • Safety Concerns: If baby is too close to an edge, seems unattended from far away, or looks like he/she may fall, flag it. If flagging for Safety Concerns, reach out to the photog's AM right away via Slack.
  • Bella Style: Images should not be overedited — only flag if editing goes completely beyond scope (teeth whitening, skin texture softening, sepia filters, etc.)
  • Image Number Sequence: Images should start at #2, not #1.
  • Image Order: Images should be in order and tell a story.
